## 🎯 ¿Qué aprenderás en el Taller 01?

- ✅ Qué es una **Red Neuronal Convolucional (CNN)** y cómo aprende
- ✅ Cargar y explorar un **dataset** de imágenes (23,000+ fotos)
- ✅ **Preprocesar** imágenes para entrenamiento
- ✅ Construir y **entrenar** tu primer clasificador de IA
- ✅ Evaluar el modelo con imágenes reales
- ✅ Comprender por qué la **calidad del dataset** lo es todo

---

## 🐙 El cierre que lo cambia todo

El taller termina con un experimento sorpresa: ¿qué pasa si le mostramos a nuestra IA (entrenada en perros y gatos)... **una imagen de un pulpo**?

> **_Garbage In → Garbage Out_**
> Una IA solo es tan buena como los datos con los que fue entrenada.

## 🚀 ¿Cómo usar este taller?

### Opción A — Google Colab (recomendado ✅)

1. Haz clic en el badge **"Open in Colab"**
2. Ve a **Entorno de ejecución → Cambiar tipo de entorno → T4 GPU**
3. Ejecuta las celdas de arriba hacia abajo, una por una

### Opción B — Local

```bash
git clone https://github.com/TU_USUARIO/TU_REPO.git
cd taller-ia
pip install -r requirements.txt
jupyter notebook Taller_IA_PerroGato_V02.ipynb
```
